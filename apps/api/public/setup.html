◆ Delta Celphei First-run setup

    Database

    Celphei needs a PostgreSQL 15+ database. Provide credentials for an account that can create the database (if it doesn't already exist).

    Administrator account

    This is the first user. They get the Admin role.

    • ○ At least 8 characters
    • ○ An uppercase letter
    • ○ A lowercase letter
    • ○ A number
    • ○ A special character
    • ○ Passwords match

    Application

    Server port and secrets. The wizard generates strong defaults; regenerate if you need to.

    Organization (optional)

    Branding shown in the UI. You can change these later under Settings → Customization.

    Polaris connection (optional)

    Lets tickets reference live asset data from Polaris.

    Directory (optional)

    Pull user accounts from Entra ID or LDAP/Active Directory.

    Mail (SMTP) (optional)

    Used for ticket notifications.

    Review & commit

    Once you click Complete Setup, the database is created (if needed), migrations run, the admin account is provisioned, and the app restarts into normal mode.
